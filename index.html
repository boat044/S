กรอก URL ของ Google Apps Script Web App เพื่อเชื่อมต่อกับ Google Sheets

  1. 1เปิด Google Sheet → Extensions → Apps Script
  2. 2วางโค้ดจากไฟล์ Code.gs ที่ได้รับมา แทนที่โค้ดเดิมทั้งหมด
  3. 3คลิก Deploy → New Deployment → เลือก Web App
  4. 4ตั้งค่า Execute as: Me และ Who has access: Anyone
  5. 5คลิก Deploy → คัดลอก URL และวางด้านล่าง
